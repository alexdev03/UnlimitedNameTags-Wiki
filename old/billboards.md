# Billboards üé•

**UnlimitedNameTags** supports four different billboard styles for displaying name tags. Each style controls how the name tag behaves relative to the player.

### Available Billboards:

- **CENTER** (Default)
- **HORIZONTAL**
- **VERTICAL**
- **FIXED**

---
